
# WordTrainerApp

Приложение для повторения английских слов.

## Сборка в Android Studio
1. Откройте папку `WordTrainerApp` в Android Studio (File → Open).
2. Дождитесь синхронизации Gradle.
3. Запустите на устройстве (Run) или соберите APK: Build → Build APK(s).

**Где править слова:** `app/src/main/assets/words.json`
